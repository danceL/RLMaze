# RLMaze


Q learning on basic Maze Environment

The RLMaze.py implements an Simple Reinforcement Learning Maze Bot Problem using Q-Learning 


## Depepndancies ##

- OpenCV (requires Microsoft Visual C++ redistributable in python3)
- Numpy
